Todos API

Use Postman to access the api

  • Show all items http://localhost:8080/api/all
  • Delete items http://localhost:8080/api/deletepost - (@RequestParam long id)
  • Add items http://localhost:8080/api/add - (@RequestParam String text)
  • Show total number http://localhost:8080/api/totalnumber
  • Show active items http://localhost:8080/api/active
  • Show completed items http://localhost:8080/api/complete
  • Change one todo http://localhost:8080/api/changeone - (@RequestParam long id)
  • Toggle all todo status http://localhost:8080/api/changeall
  • Update an items http://localhost:8080/api/update2 (@RequestParam long id, @RequestParam String text, @RequestParam boolean completed)
